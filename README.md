🚗 Proyecto Coche Autónomo con Arduino
📌 Descripción

Este proyecto consiste en el desarrollo de un vehículo autónomo basado en Arduino, construido por fases para simular un proceso real de ingeniería.

🎯 Objetivo
Controlar motores mediante Arduino
Implementar movimientos básicos
Evolucionar hacia navegación autónoma
Integrar control externo en etapas futuras
🧩 Fases del Proyecto
🔹 Fase 1 — Movimiento Básico

El coche realiza movimientos simples (avanzar, girar, etc.) mediante secuencias programadas.

🔹 Fase 2 — Movimiento Estructurado

Se implementarán movimientos por intervalos o patrones aleatorios.

🔹 Fase 3 — Evitación de Obstáculos

Uso de sensores para detectar y evitar obstáculos.

🔹 Fase 4 — Control por Giroscopio

Control del vehículo mediante una interfaz web y el giroscopio del móvil.

⚙️ Componentes
Arduino
Driver de motores
Motores DC y ruedas
Fuente de alimentación
(Futuro) Sensor ultrasónico
🚧 Estado Actual

Actualmente el proyecto se encuentra en la fase inicial de movimiento básico, trabajando en mejorar la precisión de los desplazamientos.

▶️ Uso
Cargar el código en Arduino
Conectar los componentes
Ejecutar el programa
